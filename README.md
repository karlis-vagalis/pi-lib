# @kvagalis/pi-lib

Custom set of [pi](https://pi.dev/) extensions and utilities.

## Extensions

| Extension        | Description                                                                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `exit`           | Adds an `exit` command and listens for plain-text "exit" input to gracefully shut down pi.                                                        |
| `personality`    | Adds a `personality` command, which allows to select predefined personalities to be injected in system promp.                                     |
| `resume-session` | Print short command to resume the session on exit.                                                                                                |
