# @kvagalis/pi-lib

Custom set of [pi](https://pi.dev/) extensions and utilities.

## Extensions

| Extension | Description |
|-----------|-------------|
| `exit`    | Adds an `exit` command and listens for plain-text "exit" input to gracefully shut down pi. |
| `personality`    | Adds a system prompt injector for personality of the agent. |
| `resume-session`    | Print session resuming command on exit. |
